Personal Project

Before continuing, spend some time on a personal project. It can be anything! Here are some ideas:

  • A four function calculator
  • A digital version of Tic-Tac-Toe (bonus points for an AI opponent)
  • A text based adventure game
  • A digital version of the game "Mastermind" (Google it if you haven't played it)

If you have your own idea, even better! If you're not sure how to get started or how to structure your program, I'm more than happy to help!

Remember: our program can get user input using a Scanner, and can output text using System.out.println(). You'll need to use both frequently to make an interactive application.

One hint before you start: many games and applications run until the user specifically tells the program to quit. To ensure the program keeps running, most of the code is surrounded in a while (true) { } loop, which will run forever until it hits a break statement. An example of this kind of program is given with this task. As with the last task, this won't work when the "Run" button is used, use the green play button to the left of the editor or copy it into your own project.

Please ask me questions if you're stuck or you don't know how to do something. Good luck!
