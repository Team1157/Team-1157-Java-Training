Other Operators

We've already covered most of the important operators (+, -, *, /, &&, ||, ect.), but there's some that don't fit in the other lessons or have different input and output types. We'll cover some of them now.

Assignment Operators

You've already seen one assignment operator: =. Assignment operators set the value of a variable. In addition to =, there are several more.


  a += b; // Same as a = a + b
  a -= b; // Same as a = a - b
  a *= b; // Same as a = a * b
  a /= b; // Same as a = a / b
  a++;    // Same as a += 1 or a = a + 1;
  a--;    // Same as a -= 1 or a = a - 1;

As you can see, these new assignment operators are just shorthand for commonly used pieces of code.

Comparison operators

Comparison operators allow you to compare two numbers. They always return a boolean value. The == and != operators from the last task are comparison operators. Here's the full list:


  a == b // True if a is equal to b
  a != b // True if a is not equal to b
  a > b  // True if a is greater than b
  a < b  // True if a is less than b
  a >= b // True if a is greater than or equal to b
  a <= b // True if a is less than or equal to b

Also, you can use parentheses to set the order of operation, similar to algebra


  (true || false) && true // true
  true || (false && true) // false
